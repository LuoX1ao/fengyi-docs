
# 接驳柜取件事件推送 (回调) 

### 接口说明

适用于引入无人接驳柜作为降落场，实现用户自主取件的场景。当无人机降落至接驳柜完成抛投任务时，会由丰翼回调对接方的回调地址，通知取件码信息

### 接口地址

POST
回调url


### 请求参数

| 字段名     | 属性 | 类型   | 是否必传 | 说明     |
| ---------- | ---- | ------ | -------- | -------- |
| fyCode     |      | String | 是       | 丰翼运单号|
| pickUpCode |      | String | 是       | 取件码   |

	
### 请求响应参数

| 字段名   | 属性 | 类型    | 是否必传 | 说明                      |
| -------- | ---- | ------- | -------- | ------------------------- |
| code     |      | Integer | 是       | 请求错误码 0:成功 非0:失败 |
| errorMsg |      | String  | 是       | 请求错误信息              |


### 请求示例

```json
{
    "fyCode": "FY00001",
    "pickUpCode": "301212"
}
```

### 响应示例
   
```json
{
    "code":0,
    "errorMsg":null
}
```
